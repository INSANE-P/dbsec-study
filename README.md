# 데이터베이스 보안, 이걸로 공부 끝내자 📘

> 혼자 풀고, 바로 채점하고, 약한 부분만 다시 보는 데이터베이스 보안 시험 공부 사이트.
> 그냥 링크만 누르면 바로 시작돼요.

### 👉 [지금 바로 공부하러 가기](https://insane-p.github.io/dbsec-study/)

---

## 이런 분께 좋아요

- 데이터베이스 보안 중간·기말고사를 앞두고 기출 유형으로 손풀이하고 싶은 분
- 모범답안을 눈으로만 읽다가 정작 시험장에서 백지가 되는 분
- 시간은 없는데 내가 약한 부분만 골라서 빠르게 끝내고 싶은 분

## 왜 만들었냐면

처음엔 제가 중간고사 공부하면서 그냥 재미로, 혼자 쓰려고 만든 사이트였어요.
그런데 같이 시험 보던 지인이 써보더니 "이거 너무 좋은데?" 하길래 정식으로 공개하게 됐습니다.
이번엔 기말고사 범위까지 더해서, 한 학기를 이 사이트 하나로 대비할 수 있게 만들었어요.

## 무엇을 할 수 있나요

**직접 써보고 채점하는 문제 풀이**
발문을 보고 답을 직접 적은 다음, 모범답안을 펼쳐서 비교하고 맞음·애매·틀림으로 스스로 채점해요.
눈으로 읽기만 하는 게 아니라 시험장처럼 손으로 써보는 연습이라 훨씬 오래 남습니다.

**원하는 것만 골라 푸는 필터**
중간/기말, 주차별, 그리고 유형별로 원하는 문제만 모아 풀 수 있어요. 시험 직전엔 약한 유형만 집중 공략.

**오답노트**
틀리거나 애매했던 문제만 따로 모아줍니다. 틀린 것만 다시 보는 게 제일 효율적이니까요.

**진행률·정답률 한눈에**
얼마나 풀었고 얼마나 맞혔는지 보여줘서, 어디가 부족한지 바로 감이 와요.

**주차별 정리 + 개념 사전**
시험에 나오는 핵심만 주차별로 정리해 뒀고(중요도 별점까지),
헷갈리는 약어집과 비교표, 개념 카드도 있어요. 문제 풀다 막히면 바로 옆에서 개념을 다시 잡을 수 있습니다.

## 이렇게 써보세요

1. **시험 2주 전** — 주차별 정리로 전체 흐름 한 번 훑기
2. **1주 전** — 유형별로 직접 써보며 채점하고, 약한 부분 찾기
3. **3일 전** — 오답노트로 틀린 것만 반복하기
4. **시험 당일** — 약어집·비교표로 마지막 점검

## 편하게 쓰세요

설치도 로그인도 없어요. 링크만 들어오면 끝이고, PC든 휴대폰이든 됩니다.
풀이 기록과 진행률은 그대로 저장돼서 다음에 와도 이어서 할 수 있어요. 밤샘 공부를 위한 다크 모드는 덤이고요.

## 다루는 범위

| 시험 | 범위 |

| 기말고사 | 9 ~ 14주차 — 접근제어 모델 · 암호화 · TDE·키 관리 · 빅데이터 보안·프라이버시 |
